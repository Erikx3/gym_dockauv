class gym_dockauv.objects.vehicles.BlueROV2.BlueROV2(xml_path='/home/erikx3/PycharmProjects/gym_dockauv/gym_dockauv/objects/vehicles/BlueROV2.xml', control_mode='joystick')

Bases: gym_dockauv.objects.auvsim.AUVSim

The BlueROV2 in Heavy configuration (8 T200 Thruster) by Blue Robotics, is capable of depths up to 100 metres. More information on the BlueROV2 can be found here: https://bluerobotics.com/store/rov/bluerov2/

This class is separated and written in 2 different version for the control matrix: - “direct”: - “joystick”

The parameters for the BlueROV2 are loaded via a xml file, SI units are used. The system identification data of the BlueROV2 are publicly available from - [1] Einarsson, Emil Már, and Andris Lipenitis. n.d. “Model Predictive Control for the BlueROV2,” - [2] Wu, Chu-Jou, and B Eng. n.d. “6-DoF Modelling and Control of a Remotely Operated Vehicle,”

Parameters

  • xml_path (str) – Path to xml file with parameters for BlueROV2

  • control_mode (str) – Identifier for B Matrix and u_bound [“joystick”, “direct”]

B(nu)

this function returns the control matrix \(\boldsymbol{B}\) and is individual for each AUV and thus an abstract method

When \(\boldsymbol{\tau}\) represents the external forces and \(\boldsymbol{u}\) the control input, then

\[\boldsymbol{\tau} = \boldsymbol{B}_{6\times a} \boldsymbol{u}_{a\times 1}\]

Where the dimension \(\boldsymbol{a}\) is the number of actions available for the system :rtype: ndarray :return: array 6xa
